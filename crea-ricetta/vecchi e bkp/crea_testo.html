Come creare il tuo elenco di materie prime:

  • Compila l'area di testo sottostante seguendo l'esempio. Ovvero, per ogni riga UN solo ingrediente.
  • E' importante che ogni riga contenga solo il nome di un ingrediente. Vai a capo e inserisci un altro nome.
  • Quando sei pronto, clicca su SALVA. Verrà chiesto di salvare un file di testo.
  • Questo file potrai utilizzarlo in Crea Ricetta per avere suggerimenti sulle materie da te possedute.
  • Puoi anche modificare questo file direttamente dal tuo pc. L'importante è seguire la regola di una riga - un ingrediente e che il file sia un file di testo TXT.
